Wine

Instalar Wine

Capa de compatibilidad necesaria para ejecutar juegos y emuladores de Windows en Abalar 11 y 12.

¿Qué es Wine y para qué sirve?
Wine es una capa de compatibilidad que permite ejecutar programas y juegos diseñados para Windows directamente en Linux. En Abalar 11 y 12, Wine es imprescindible para poder abrir los juegos y emuladores de esta plataforma. Sin Wine instalado, los archivos .exe no se ejecutarán.
// Paso previo

Permisos de administrador

Para instalar Wine es necesario tener permisos de administrador en el equipo Abalar. Si aún no los tienes, puedes conseguirlos siguiendo los pasos de Abalar Exploits.

Sin permisos de administrador no podrás ejecutar los comandos necesarios para la instalación.
// Paso 1

Si tienes las actualizaciones automáticas desactivadas

Si en tu equipo se han desactivado las actualizaciones automáticas (bloqueando el archivo /etc/hosts), antes de continuar ejecuta este comando para que apt pueda conectarse a internet:

bash — ejecutar como root
chattr -i /etc/hosts
Al acabar la instalación de Wine puedes volver a bloquearlo con: chattr +i /etc/hosts
// Paso 2

Abre la consola como root

Abre un terminal y accede como superusuario:

bash
su -

Se te pedirá la contraseña de root. Una vez dentro, el prompt cambiará a #.

// Paso 3

Instalar Wine

Ejecuta los siguientes comandos en la consola de root:

bash — ejecutar en orden
dpkg --add-architecture i386 wget -NP /etc/apt/sources.list.d/ https://dl.winehq.org/wine-builds/debian/dists/bookworm/winehq-bookworm.sources echo "deb http://deb.debian.org/debian bookworm main contrib non-free non-free-firmware" > /etc/apt/sources.list.d/debian-oficial.list apt update apt install --install-recommends winehq-stable
La descarga puede tardar varios minutos. No cierres la terminal hasta que finalice.
// Paso 4 (opcional)

Restaurar el bloqueo de actualizaciones

Si ejecutaste el Paso 1, vuelve a bloquear el archivo hosts:

bash — ejecutar como root
chattr +i /etc/hosts
Wine ya está instalado. Ahora puedes ejecutar los juegos y emuladores de STRECH Hub.
Cómo abrir los juegos

Con Wine instalado, abrir cualquier juego o emulador es muy sencillo.

// Paso 1

Entra a la carpeta del juego

Abre el gestor de archivos y navega hasta la carpeta donde descomprimiste el juego. Dentro encontrarás un archivo .exe (normalmente tiene el nombre del juego o se llama launcher.exe, game.exe o similar).

// Paso 2

Abre el .exe con Wine

Haz clic derecho sobre el archivo .exe y selecciona Abrir con → Wine Windows Program Loader. El juego arrancará como si estuvieras en Windows.

Si no aparece la opción, también puedes abrir un terminal en esa carpeta y ejecutar: wine nombre_del_juego.exe
// Alternativa por terminal

Abrir desde terminal

Si prefieres usar la terminal, navega hasta la carpeta del juego y ejecuta:

bash — ejemplo
wine nombre_del_juego.exe

Sustituye nombre_del_juego.exe por el nombre real del ejecutable.
